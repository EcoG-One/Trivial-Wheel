# Trivial Wheel #

## A comprehensive trivia wheel game in HTML, CSS, Javascript.
### A fully functional multiplayer quiz game with colorful spinning wheel, question bank, score tracking, and win conditions. 

## Gameplay Features:

* 6 colored wheel slices matching 6 trivia categories (Geography/blue, Entertainment/pink, History/yellow, Art & Literature/purple, Science & Nature/green, Sports & Leisure/orange)
* Fixed red pointer at the 12 o'clock position that stays stationary while the wheel spins
* 110 multiple-choice questions per category (660 total questions - feel free to add more :))
* Multiplayer support - add 2-6 players with customizable names
  
## Scoring System:

- Each player collects colored dots representing categories they've answered correctly

- Visual scoreboard shows each player's collected colors

- Current player is highlighted with a yellow ring

## Game Flow:

Correct answers: confetti celebration, color added to player's collection, same player continues

Wrong answers: next player's turn

When a player collects all 6 colors, they face a Final Question from a random category

Win the final question = victory with celebration! 

Lose = lose a color and next player goes

## Visual Polish:

Smooth wheel spinning animation with realistic deceleration

Flip animations for the question panel

Confetti effects for correct answers and winning

Responsive layout - stacks vertically on mobile

Beautiful gradient background with glassmorphism UI
